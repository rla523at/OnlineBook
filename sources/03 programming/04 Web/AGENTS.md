# 04 Web 문서 작성 규칙

이 규칙은 `sources/03 programming/04 Web` 디렉터리와 그 하위 디렉터리의 문서 작성 및 수정에만 적용한다.

## 기준 독자

- 이 디렉터리의 문서는 Web 개발 초심자를 기준 독자로 한다.
- backend, database, deployment, authentication, API를 다루는 문서에서는 해당 주제의 핵심 용어를 처음부터 알고 있다고 가정하지 않는다.
- Python 예제가 있는 문서는 Python 기본 문법을 읽을 수 있는 독자를 기준으로 삼을 수 있다. 단, Web request/response, DB, ORM, migration, deployment 용어는 문서 흐름에 필요한 만큼 설명한다.

## 내부 링크

- Web 하위 문서 사이에 선행 지식 관계가 있으면 관련 문서를 연결한다.
- 내부 링크를 추가할 때는 링크 대상이 어떤 배경을 보충하는지 현재 문서에서 한 문장으로 설명한다.
